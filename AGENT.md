# AGENT.md

This file provides guidance to LLM agents when working with code in this repository.

## Repository Overview

This is a python LangGraph application containing two main software components:

**Component 1: LangGraph Agent** - The core agent containing logic to triage inbound support tickets
**Component 2: Evals** - LangSmith evaluations to execute and validate against the LangGraph Agent

## Component 1: LangGraph Agent

- `simple_agent` - LangGraph python agent

## Component 2: Evals

- 'eval': LangSmith evaluations
